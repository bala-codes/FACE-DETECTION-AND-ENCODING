# FACE DETECTION & ENCODING.
A simple project which detects the faces from a photo and encodes the detected faces into 128 numbers.
This project is an excerpt from my face recognition course from Lynda Learning taught by Adam Geitgey.

## Run yourself at [Google Colab](https://colab.research.google.com/github/bala-codes/FACE-DETECTION-AND-ENCODING/blob/master/Face%20Detection.ipynb) and .py file at [here](https://github.com/bala-codes/FACE-DETECTION-AND-ENCODING/blob/master/Face%20Detection.py)
